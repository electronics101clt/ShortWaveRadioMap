ShortWave Radio Map

⚠️ No Internet Connection

This app needs an active internet connection to access live shortwave radio receivers around the world.

What This App Does

ShortWave Radio Map connects you to a global network of KiwiSDR and WebSDR receivers - real shortwave radio receivers (1-30 MHz) operated by radio enthusiasts worldwide. You can tune into live broadcasts, explore the radio spectrum, and listen to stations from across the globe.

How to Get Connected

  • Check WiFi: Ensure your device is connected to a WiFi network with internet access
  • Router Connection: Verify your WiFi router has an active internet connection
  • Mobile Hotspot: Connect to a phone's mobile hotspot if WiFi is unavailable
  • Network Restrictions: If in a restricted network environment, you may need:
    • VPN service (Astrill, ExpressVPN, Surfshark)
    • Proxy or circumvention tools (Shadowsocks, V2Ray)
    • Mobile hotspot from a device with these tools

About SDR Receivers

This app is a wrapper for online SDR (Software Defined Radio) maps that show real radio receivers. When you connect, you'll see a world map where you can click on any receiver to tune into that location's radio spectrum in real-time.

Supported Services:

rx.skywavelinux.com • map.kiwisdr.com • rx-tx.info

Frequency Range: 1-30 MHz (Shortwave HF)
